🔗 n8n Webhook — แจ้งเตือนกลุ่ม Line OA
วาง Webhook URL จาก n8n ด้านล่าง — เมื่อ Sales ยกเลิก/เลื่อนนัด ระบบจะเรียก n8n ทันที แล้ว n8n ส่งข้อความเข้ากลุ่ม Line OA
🛈 กรณีที่จะแจ้งเตือน
❌
Sales ยกเลิกนัดหมาย
status เปลี่ยนเป็น "ยกเลิก" — ส่งข้อมูลไป n8n พร้อม type: cancel
📅
Sales เลื่อนนัดหมาย
วันที่นัดเปลี่ยน — ส่งข้อมูลพร้อม type: postpone, oldDate, newDate
🗑
Sales ลบนัดหมาย
ลบ booking ออกจากระบบ — ส่งข้อมูลพร้อม type: delete
📝 ขั้นตอนสร้าง n8n Workflow (ดูคู่มือ PDF)
  1. สมัคร n8n.io (ฟรี) → สร้าง workflow ใหม่
  2. เพิ่ม node Webhook → Copy URL → วางในช่องด้านบน
  3. เพิ่ม node Line → ใส่ Channel Access Token → เลือก Push Message
  4. ตั้งค่า message ใช้ข้อมูลจาก webhook: {{$json.hospital}} เป็นต้น
  5. กด Activate → กลับมากด "ทดสอบส่ง" ในหน้านี้

firebase functions:secrets:set LINE_CHANNEL_TOKEN
🛈 กรณีที่จะส่งแจ้งเตือน
❌
Sales ยกเลิกนัดหมาย
เมื่อ status เปลี่ยนเป็น "ยกเลิก" — ส่ง Flex Message สีแดงเข้ากลุ่ม
📅
Sales เลื่อนนัดหมาย
เมื่อวันที่นัดเปลี่ยน — ส่ง Flex Message แสดงวันเดิม → วันใหม่
🗑
Sales ลบนัดหมาย
เมื่อลบ booking ออกจากระบบ — ส่งแจ้งเตือนเข้ากลุ่ม
📄 ขั้นตอน Deploy (ดูไฟล์ DEPLOY_GUIDE.md)
1 หา Channel Access Token จาก Line Developers Console → Messaging API
2 เพิ่ม OA เข้ากลุ่ม Line → รับ Group ID จาก webhook event
3 firebase functions:secrets:set LINE_CHANNEL_TOKEN
4 firebase functions:secrets:set LINE_GROUP_ID
5 firebase deploy --only functions
